# HR-Employee-Attrition-Analysis

This repository contains a data analysis and machine learning project focused on Employee Attrition. The goal of this project is to uncover the underlying factors that cause employees to leave a company and to identify patterns that can help HR departments improve retention rates.

Replacing an employee can cost a company significantly in recruitment, onboarding, and lost productivity. By leveraging data, we can predict which employees are at the highest risk of leaving and take proactive measures to retain top talent.
